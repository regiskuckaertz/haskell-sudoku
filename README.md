# haskell-sudoku

1. Install [stack](https://docs.haskellstack.org/en/stable/README/)
2. Build with `stack build`
3. Play with `stack ghci`
4. Run with `stack exec haskell-sudoku`
5. Profile with `stack exec haskell-sudoku +RTS -p`.
